## ภาพรวม

นี่คือ Portfolio ส่วนตัวของผม — เว็บที่คุณกำลังดูอยู่ตอนนี้ สร้างตั้งแต่ต้นด้วย Vue 3, Tailwind CSS v4 และ Color System แบบ Material Design 3 Dark

## Design System

Palette สีเป็นไปตาม Material Design 3 Dark Scheme โดยใช้ Cyan (`#81ecff`) เป็น Primary Accent และ Pink (`#ff3399`) สำหรับ Neon Glow Effect Typography ใช้ Font 3 ตัว: Chakra Petch สำหรับ Heading, Manrope สำหรับ Subheading และ Inter สำหรับ Body Text

## ฟีเจอร์หลัก

- 🌧️ **Matrix Rain** — Background Animated บน Canvas
- 📌 **Sticky Navbar** — Highlight Section Active ตาม Scroll
- 🧩 **Bento Grid** — Layout การ์ดแบบ Asymmetric สำหรับส่วน Personal, Skills
- 🌐 **i18n** — สลับภาษา EN / TH ผ่าน vue-i18n
- 💬 **Project Modal** — Hover เพื่อ Preview + คลิกเพื่อเปิดพร้อม Markdown Body

## Tech Stack

| Layer | Technology |

| Frontend | Vue 3 · Tailwind CSS v4 |
