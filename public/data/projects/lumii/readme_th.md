## ภาพรวม

lumii คือเว็บแอปทดสอบบุคลิกภาพ MBTI ที่ออกแบบมาอย่างสวยงาม พาผู้ใช้ผ่านชุดคำถามที่คัดสรรมาด้วย Transition Animated ที่ลื่นไหล จากนั้นเปิดเผยผลลัพธ์ MBTI 4 ตัวอักษรพร้อม Result Screen ที่ขัดเกลามาอย่างดี

## ฟีเจอร์

- ✨ Transition ระหว่างคำถามแบบ Animated ที่ลื่นไหล
- 📊 Progress Indicator แสดงความคืบหน้า
- 🎨 Result Screen เฉพาะสำหรับแต่ละ Type พร้อม Color Palette ไม่ซ้ำกัน
- 📱 Responsive เต็มรูปแบบ — ใช้ได้ทั้ง Mobile และ Desktop
- ⚡ เร็ว — ไม่มี Backend คะแนนคำนวณฝั่ง Client ล้วน
