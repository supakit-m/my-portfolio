## Overview

lumii is a beautifully designed MBTI personality test web app. It guides users through a curated set of questions with smooth animated transitions, then reveals their 4-letter MBTI type with a polished results screen.

## Features

- ✨ Smooth question-to-question animated transitions
- 📊 Progress indicator showing how far along you are
- 🎨 Per-type result screens with unique color palettes
- 📱 Fully responsive — works on mobile and desktop
- ⚡ Fast — no backend, pure client-side scoring
