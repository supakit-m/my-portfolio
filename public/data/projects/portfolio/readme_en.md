## Overview

This is my personal developer portfolio — the site you're looking at right now. Built from scratch with Vue 3, Tailwind CSS v4, and a Material Design 3 inspired dark color system.

## Design System

The color palette follows Material Design 3 dark scheme with a cyan (`#81ecff`) primary accent and pink (`#ff3399`) brand-alt for neon glow effects. Typography uses three complementary typefaces: Chakra Petch for headings, Manrope for subheadings, and Inter for body text.

## Key Features

- 🌧️ **Matrix Rain** — Canvas-based animated background
- 📌 **Sticky Navbar** — Scroll-aware active section highlighting
- 🧩 **Bento Grid** — Asymmetric card layouts for Personal and Skills sections
- 🌐 **i18n** — EN / TH language switching via vue-i18n
- 💬 **Project Modal** — Hover-to-preview + click-to-open with full Markdown body
